# KuCoin-Rates-Bot

Unoffical KuCoin-Rates-Bot by dmitriypy

**Installation**
-----

    git clone https://github.com/dmitriypy/KuCoin-Rates-Bot.git


List of commands
-----

    /start@Username
    /price@Username <pair>
    /rates@Username <pair> num[min|hour|day|week]
